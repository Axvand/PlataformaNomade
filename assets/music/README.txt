# Pasta de Músicas

Coloque seus arquivos MP3 nesta pasta para serem carregados como músicas iniciais na aplicação.

Por exemplo:
- track1.mp3
- track2.mp3
- track3.mp3

Lembre-se de atualizar a configuração em `script.js` para apontar para esses arquivos e mostrar os metadados corretos.

Nota: Neste diretório você também pode criar pastas organizadas por álbum ou artista, se preferir.
